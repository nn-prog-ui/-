# AI FX市場監視システム 使い方ガイド

> **重要なお知らせ**
> このシステムは「価格の分析と参考情報の提示」だけを行います。
> **自動で注文を出す機能はありません。**
> 最終的な売買の判断は、必ずあなた自身が行ってください。

---

## このシステムは何をするの？

為替（FX）の価格を自動で分析して、

- 📈 **買い候補** ── 上がる可能性が高い状況のとき
- 📉 **売り候補** ── 下がる可能性が高い状況のとき
- ⏸ **見送り** ── どちらとも言えない・リスクが高い状況のとき

の3種類に分けて画面に表示します。

対応している通貨ペアは **8種類**：
`USD/JPY`・`EUR/USD`・`GBP/USD`・`EUR/JPY`・`AUD/JPY`・`AUD/USD`・`EUR/GBP`・`GBP/JPY`

## 起動方法

```bash
# 1. 必要なライブラリをインストール（初回だけ）
pip3 install -r requirements.txt

# 2. サーバーを起動
python3 -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 3. ブラウザで開く
# http://localhost:8000
```

## 毎日の使い方（3ステップ）

### ステップ 1：ブラウザで開く
`http://localhost:8000` にアクセスすると **判定ページ** が表示されます。

### ステップ 2：判定を確認する
画面の上部に大きく「買い候補」「売り候補」「見送り」のどれかが表示されます。

### ステップ 3：自分で判断して記録する
画面を最後まで読んで納得できたら「買いで記録」「売りで記録」「見送りで記録」ボタンを押します。
**ボタンを押しても実際の取引は発生しません。** 判断履歴が記録されるだけです。

## 画面の見方

### 📊 判定ページ（`/`）← メインページ

```
┌─────────────────────────────────┐
│  USD/JPY        現在価格: 152.340 │
│  1時間ごとのデータを使用          │
└─────────────────────────────────┘
        ↓
┌─────────────────────────────────┐
│  判定: 買い候補                   │
│  総合スコア: +7                  │
│  ↑これは全7条件をすべて満たした    │
└─────────────────────────────────┘
        ↓
┌─────────────────────────────────┐
│  条件の充足数                     │
│  ▲ 買い  7/7 ■■■■■■■         │  ← 7条件すべて満たしている
│  ▼ 売り  1/7 ■□□□□□□         │  ← 売り条件は1つしか満たしていない
└─────────────────────────────────┘
```

**スコアの読み方：**
| スコア | 意味 |
|---|---|
| +7 | 買い条件7つすべて満たした → 「買い候補」 |
| +5〜+6 | 惜しいが条件不足 → 「見送り」 |
| 0 | どちらでもない → 「見送り」 |
| -5〜-6 | 売り条件が多め → 「見送り」 |
| -7 | 売り条件7つすべて満たした → 「売り候補」 |

> **ポイント：** 7つ全部揃ったときだけ買い・売りと判定されます。6/7でも「見送り」です。

### 📋 各セクションの説明

#### 相場の流れ
| 表示 | 意味 |
|---|---|
| 上昇 | 価格が上がる方向 |
| 下降 | 価格が下がる方向 |
| 横ばい | あまり動いていない |

- **1日単位** → 大きな流れ（長い目で見た方向）
- **4時間単位** → 中くらいの流れ
- **1時間単位** → 直近の状態（高値を超えた / 安値を下回った）

#### 価格分析の数値

| 項目 | 意味 | 目安 |
|---|---|---|
| 上がりすぎ・下がりすぎ（RSI） | 0〜100の数値 | 70以上＝買われすぎ、30以下＝売られすぎ |
| 値動きの激しさ | 最近の値動きの大きさ | 「通常」が安全、「高い」は注意 |
| 最近の高い価格 | 直近20時間の最高値 | この価格を超えると「上昇突破」 |
| 最近の安い価格 | 直近20時間の最安値 | この価格を下回ると「下降突破」 |

#### 価格の上限・下限（ボリンジャーバンド）
統計的に「このくらいの範囲で動くだろう」という上限・下限を示した帯です。
- **上限に近い** → 反落しやすい水準
- **下限に近い** → 反発しやすい水準

#### 上昇・下降の勢い（MACD）
短期と長期の平均価格の差を見て、今の勢いを判断します。
- **上昇に転換** → 下から上に勢いが変わった
- **下降に転換** → 上から下に勢いが変わった

#### 価格パターン（直近の値動きの形）
ローソク足チャートの形から「上がりやすい形か、下がりやすい形か」を自動判定します。
★の数が多いほど信頼性が高いとされます（あくまで参考）。

#### 取引の目安価格
| 項目 | 意味 |
|---|---|
| 買う・売る価格の目安 | このあたりで取引を検討する価格 |
| 損失を止める価格 | ここまで逆に動いたら撤退する価格（損切りライン） |
| 利益確定の目標価格 | ここまで動いたら利益を確定する価格 |
| 損失と利益の比率 | 例）2.0 = 損失の2倍の利益を狙う設定 |

> **例：** 損失が1万円なら、利益目標は2万円（比率2.0）

### 📜 履歴ページ（`/history`）
過去に「買いで記録」「売りで記録」「見送りで記録」した内容を一覧で確認できます。
検索バーで通貨ペアや日付を絞り込めます。

### 📈 成績ページ（`/performance`）
過去の取引記録から「勝率」「損益合計」などを集計して表示します。
実際の口座の成績ではなく、このシステム上の記録の集計です。

### ⚙️ 設定ページ（`/settings`）
- 自動スキャンの間隔（何分ごとに分析するか）
- メール通知のON/OFF
- Claude AI APIキーの接続確認

## データソースの選び方

`.env` ファイルの `DATA_SOURCE=` で切り替えます。

| 設定値 | 使う場面 | 特徴 |
|---|---|---|
| `csv` | デフォルト | CSVファイルを読む。安全・安定 |
| `yfinance` | 口座不要でリアルタイム分析したいとき | Yahoo Financeから無料取得 |

### yfinance（ヤフーファイナンス）を使う手順

```bash
# 1. ライブラリをインストール
pip3 install yfinance

# 2. .env ファイルを編集
DATA_SOURCE=yfinance  # ← csv から変更する

# 3. サーバーを再起動
```

## Claude AI コメントを使う手順

設定しなくても「ルールベース自動生成」でコメントは表示されます。
より自然な文章にしたい場合は Claude AI を接続できます。

```
1. https://console.anthropic.com/ でアカウントを作成（無料）
2. 「API Keys」でキー（sk-ant-...）を発行
3. .env ファイルに追記:
   ANTHROPIC_API_KEY=sk-ant-ここにキーを貼り付ける
4. サーバーを再起動
5. /settings → 「接続テスト」ボタンで確認
```

## よくある質問

**Q. ボタンを押したら本当に注文されませんか？**
→ はい、されません。SQLiteというデータベースに「記録した」という履歴が保存されるだけです。

**Q. CSVファイルがなくてもいいですか？**
→ はい。`DATA_SOURCE=yfinance` に設定すれば Yahoo Finance からリアルタイムに取得できます。

**Q. スコア +6 でも「見送り」になるのはなぜ？**
→ このシステムは安全を優先して「全7条件が揃ったとき**だけ**」を買い・売りと判定します。6/7では「惜しい」ですが「見送り」です。

**Q. 判定は何分ごとに更新されますか？**
→ ページを開くたびに最新データで分析します。自動スキャンは `/settings` で間隔を設定できます（デフォルト60分）。

**Q. スコアの数字が地政学で変わるのはなぜ？**
→ 世界情勢（戦争・経済制裁など）が USD に大きく影響するとき、スコアを±1 補正します。あくまで参考です。

## 安全に関する原則

1. **自動売買しない** ── ボタンを押しても注文は出ない
2. **デモ口座専用** ── 本番口座への接続機能は実装していない
3. **APIキーはファイルに書かない** ── すべて `.env` ファイルで管理
4. **判断は人間が行う** ── AI・システムの判定はあくまで参考情報
